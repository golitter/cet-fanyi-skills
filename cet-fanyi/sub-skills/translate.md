# 翻译解析子技能（translate）

## 核心原则

**先保证"意思对 + 语法对 + 写得完"，再追求"词更准 + 句更漂亮"。**

## 工作流

### Step 1：分析句子结构，确定主干

勾画句子的主干部分（主语 + 谓语 + 宾语），明确核心逻辑。

操作要点：
- 把所有定语（"……的"）和状语（"……地"、时间、地点等）先剥离
- 只看骨架，确认主干成分是否完整

### Step 2：翻译主干部分

将主干（主语、谓语、宾语）准确翻译成英文。

要求：
- 核心意思表达清晰、语法正确
- 这一步完成，句子的"得分主体"已经拿到

### Step 3：补充修饰成分

把 Step 1 剥离的修饰成分逐个翻译，再填入句子中。

修饰成分类型：
- 时间状语（长久以来 → For a long time）
- 地点状语（在…… → in/at...）
- 定语（……的 → adjective / prep. phrase / relative clause）
- 程度副词（极大地 → greatly / significantly）

### Step 4：通读润色

将完整句子通读一遍，检查：
- 语法是否正确（时态、单复数、冠词）
- 语序是否符合英语习惯
- 有无拼写错误

## 输出格式

将原文按句号（。）拆分为独立句子，每句逐一解析。

```
## 原文
<中文原文>

---

### 句子 1：<摘录第一个完整句子>

**提取主干**
- 主干：<主语> + <谓语> + <宾语>
- 修饰成分：<列出剥离的修饰成分>

**翻译主干**
<主干英文翻译>

**补充修饰**
- <修饰成分 1 中文> → <英文翻译>（说明为什么这样翻）
- <修饰成分 2 中文> → <英文翻译>（说明为什么这样翻）

**完整译文**
<该句完整英文翻译>

**讲解**
<对该句的翻译要点、易错点、词汇用法等进行讲解>

---

### 句子 2：<摘录第二个完整句子>
（结构同上）
```
