# 翻译应试技巧子技能（xy-tips）

## 核心策略

四六级翻译考察的是"把意思表达清楚"，不是"词汇量比拼"。遇到不会的词，用以下三条策略化险为夷。

**策略一：不会的词用上位词替代** — 把具体名词替换为它所属的大类词。上位词一定是你确定会拼、会用的常见词，宁可宽泛也不写错。

**策略二：用会的词把意思描述出来** — 遇到动词/形容词不会翻，用简单词组成句子把意思解释出来（释义法 paraphrase）。多用简单句、并列句，少纠结一个高级词。只要意思到了，同样得分。

**策略三：大词瞎胡写，小词装没看见** — 大词（核心名词、动词）是句子意思支柱，不能空，不会就用策略一二或写拼音；小词（程度副词、修饰形容词、补充说明）删掉不影响主干，果断跳过。优先保大词。

## 工作流

### Step 1：确定目标句子

按以下优先级获取需要处理的中文句子：

1. 本次命令的 query（`/cet-fanyi xy-tips <句子>`）
2. 若 query 为空，取会话上下文中最近一次翻译练习的中文原文

如果都取不到，提示用户输入中文句子。

### Step 2：标注难点词

扫描目标句子，标记出四六级考生可能不会翻译的词或短语。对每个难点词，判断适用哪条策略：

- 具体名词 → 策略一（上位词替代）
- 动词/形容词/短语 → 策略二（释义描述）
- 修饰性小词 → 策略三（跳过）

### Step 3：逐词演示

对每个难点词，按策略给出替代方案：

```
**<难词>** → <策略名>
- 上位词/释义：<替代英文>
- 思路：<为什么这样替代>
```

### Step 4：给出最终译文

综合所有替代，输出去掉小词后的完整英文翻译。

## 输出格式

```
## 目标句子
<中文原文>

---

### 难点标注
- <难词 1>（<词性>）→ 策略 X：<策略名>
- <难词 2>（<词性>）→ 策略 X：<策略名>
- ...

---

### 逐词演示

**<难词 1>** → 策略一：上位词替代
- 替代：<英文上位词>
- 思路：<说明>

**<难词 2>** → 策略二：释义描述
- 释义：<英文句子>
- 思路：<说明>

**<难词 3>** → 策略三：跳过
- 原因：<为什么可以跳过>

---

### 最终译文
<完整英文翻译>
```
